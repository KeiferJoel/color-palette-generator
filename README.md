# 🎨 Color Palette Generator

> **Generate beautiful color palettes instantly.**
> A modern and responsive web application built with **HTML, CSS, and JavaScript** to help designers and developers quickly discover inspiring color combinations.

---

## ✨ Features

* 🎲 Generate **5 random color palettes** instantly.
* 📋 Copy HEX color codes with a single click.
* 🌙 Toggle between **Light** and **Dark** themes.
* 🌈 Animated gradient title for a polished interface.
* ✨ Smooth UI transition animations.
* 📱 Fully responsive design.
* ⚡ Lightweight and fast.
