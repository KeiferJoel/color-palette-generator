# Color Palette Generator

A simple web application that generates random color palettes and allows users to copy HEX color codes with a single click.

## Features

* Generate 5 random colors
* Copy HEX codes to the clipboard
* Responsive and clean UI
* Built with pure HTML, CSS, and JavaScript

## Technologies

* HTML5
* CSS3
* JavaScript (ES6)

## How to Run

Open `index.html` in your browser.
